Pour lancer le front : 

-npm install

-ng serve 

Il faut avoir la version v20.10.0 de Node Js 


Pour lancer le backend (Avec InteliJ) : 

- charger le fichier pom.xml

- (Possiblement) setup le SDK à 17 de java

- Dans application.properties : mettre les informations de son user de la bdd (user et password)

- Creer un schéma nommé  qlogiciel (ou modifier l'url dans application.properties pour viser votre schéma)

- Importer la bdd avec le fichier DumpDB.sql

- Lancer DemoApplication
